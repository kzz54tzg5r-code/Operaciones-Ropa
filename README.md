# ORION APP13 - Proyecto, colaboradores y meta

Cambios:
- Resumen Ejecutivo usa sólo tiendas seleccionadas del proyecto.
- Conversión y Recuperación Económica conservan todas las tiendas.
- Día Anterior agrega No. Colaboradores, Meta Colaboradores, Productividad Total y Diferencia vs Meta.
- Colaboradores se unifican con hoja plantilla cuando existe.
- Ranking por colaborador agrega Meta Colaborador y Diferencia vs Meta cuando aplica.
