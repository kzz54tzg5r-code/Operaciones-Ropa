# ORION PRO v3

Plataforma Indicadores de Recuperación de Mercancía — Operaciones Ropa.

## Archivos principales

- `app.py`: entrada principal para Streamlit Cloud.
- `requirements.txt`: dependencias.
- `modules/`: funciones de carga, normalización, KPIs y diseño.
- `pages/`: pestañas del tablero.
- `config/metas.json`: metas iniciales.

## Despliegue en Streamlit

1. Sube el contenido de esta carpeta al repositorio de GitHub.
2. En Streamlit Cloud selecciona el repositorio.
3. Main file path: `app.py`.
4. Deploy.

## Importante

No subir archivos `.pyc`, `__pycache__` ni archivos llamados `download`.
