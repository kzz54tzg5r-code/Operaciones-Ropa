# ORION APP13 - Config proyecto, colaboradores e índice

Cambios:
- Configuración de Metas permite seleccionar tiendas del proyecto.
- El filtro de tiendas del proyecto aplica a todas las pestañas excepto:
  Conversión, Recuperación Económica y Ranking de Tiendas.
- Colaboradores se homologan con hoja plantilla.
- Productividad se agrupa por Tienda + Colaborador.
- Índice Integral agrega vista por colaborador.
